Reflection

My Lab 7 page was producing two prompt windows before the first could be sorted. To fix this, I reviewed my code and realized that I was calling my function twice, once in my function and once in my document writeln section. To fix it, I created a new variable named “sorted” and set it to the function. After that, I replaced the second function call in the writeln section with the new variable. I tested in the console and it says that “sorted” was defined in line one, column one of my code, which is confusing because that space is occupied by the title of the lab in comment form. I am counting it as debugged, though, because it is working properly now, despite what the console is whining about.
